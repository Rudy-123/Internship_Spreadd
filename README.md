# Spreadd Internship Projects

This repository contains web development projects completed during the internship program - covering frontend development with React, backend APIs with Express.js, CSS techniques, and full-stack applications. Projects include LoginLab (JWT authentication), User Management System (REST API), and VibeCart (complete e-commerce platform with MongoDB, React frontend, and admin dashboard).

The code follows modern JavaScript practices, includes proper error handling, and demonstrates full-stack development concepts from database integration to responsive design. Each project is organized in its own folder with clear structure and documentation.
