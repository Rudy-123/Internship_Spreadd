The complete flow :-- 
Client Request -> Route Check -> Middleware Process -> Controller -> Service -> DB -> Response Send

The Folders:--
src/config :- sets the app with the environment
src/routes :- routes the URL to the Controller
src/middleware :- processes the request 
src/controllers :- extract the data and process the request and send the response 
src/services :- data processing, calculations and external api calls    
src/models :- DB schema
